
TaskNova 🚀

TaskNova is a modern full-stack task management web application designed to help users efficiently create, organize, manage, and track daily tasks. The platform provides a clean and user-friendly dashboard where users can monitor task progress, update task status, and improve productivity seamlessly.

🌐 Live Demo

🔗 Live Website:
https://web-production-17582.up.railway.app/

📌 Features

✅ User Authentication (Login & Signup)
✅ Create New Tasks
✅ Edit & Update Tasks
✅ Delete Tasks
✅ Track Task Progress
✅ Responsive Dashboard UI
✅ Full-Stack MERN Application
✅ Secure Backend APIs
✅ Real-Time Task Management Experience

⚙️ Installation & Setup Guide
1️⃣ Clone Repository
git clone https://github.com/tanyasharma04/TaskNova
2️⃣ Navigate to Project Folder
cd TaskNova
3️⃣ Install Dependencies
Frontend Setup
cd frontend
npm install
npm run dev
Backend Setup
cd backend
npm install
npm start
🔑 Environment Variables

Create a .env file inside the backend folder and add:

MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_secret_key
PORT=5000
